## 標準出力

`print`関数を実行

```python
print('Hello World')
# Hello World
```

## 数値の型

int型とfloat型。型は`type`関数で求められる。

```python
print(type(1))
# <type 'int'>

print(type(1.0))
# <type 'float'>
```

```python
print(1 + 1.0)
## 2.0

print(1.0 - 1)
## 0.0

print(10 / 3)
# 3

print(10.0 / 3)
# 3.33333333333
```
